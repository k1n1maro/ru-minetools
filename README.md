- **Переводит FTB квесты** - Все задания и описания станут понятными

- **Помнит переводы** - Кэширует результаты для скорости
- **Сама обновляется** - Скачивает новые версии автоматически

## 🚀 Как начать пользоваться

### Шаг 1: Скачайте программу
Нажмите на большую зеленую кнопку "СКАЧАТЬ EXE" выше ⬆️

### Шаг 2: Запустите программу
Двойной клик по скачанному файлу

### Шаг 3: Получите код доступа
1. Подпишитесь на [канал в Telegram](https://t.me/ruquestbook)
2. Напишите [боту](https://t.me/ru_minetools_auth_bot)
3. Введите полученный код в программе

### Шаг 4: Переводите!
- Для квестов: выберите папку с `.minecraft/config/ftbquests`
- Для модов: выберите папку с `.minecraft/mods`

## 🔄 Как обновляться

**Программа обновляется автоматически!**

1. **При запуске** программа проверит обновления
2. **Если есть новая версия** - появится уведомление
3. **Нажмите "Обновить"** - программа все сделает сама
4. **Готово!** Программа перезапустится с новой версией
